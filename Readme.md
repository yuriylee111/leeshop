# Lee Shop Project

## Launch instruction

1. Start mysql server 
2. Paste script from leeshop.sql (src/db/ )
3. Execute all
4. Open IDEA and launch web application using Tomcat (my version 9.0.62)
5. Open browser: `http://localhost:8080`
